
# COVID-19 Dashboard

 A simple web application that displays COVID-19 data by country using API.

 ## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap
- Chart.js
- COVID API
- Country Flags API
